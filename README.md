# HTTP-сервис для библиотеки
## Описание проекта.
Данный проект - сервис для управления библиотекой. Он представляет собой RESTful API-сервис, работающий с БД SQLite.  
Основные возможности:
1) Управление книгами - добавление, обновление данных и получение списка книг;
2) Управление читателями - регистрация читателей, получение списка книг по конкретному пользователю;
3) Библиотечные операции - выдача и возврат книг;
4) Логирование - все операции и действия логируются в формате JSON.

## Как работать?
1) Склонируйте репозиторий и перейдите в папку проекта:  
```cd library_server```
3) Создайте файл окружения на основе примера:  
```cp .env.example .env```
4) Выполните установку зависимостей:  
```go mod tidy```
5) Запуск проекта:  
```go run cmd/main.go```

После сервер будет доступен по адресу http://localhost:8080, или по ссылке, предоставленной GitHub Codespaces.

## Примеры curl-запросов.
Необходимо URL заменить на настоящий адрес сервера, а ID на настоящие UUID из ответов сервера.  

Добавление книги:  
```curl -X POST URL/books -H "Content-Type: application/json" -d "{\"title\": \"Преступление и наказание\", \"author\": \"Федор Достоевский\", \"isbn\": \"9785170906308\", \"year\": 1866}"```

Получение список всех книг:  
```curl URL/books```

Получение информации книги по ID:  
```curl URL/books/ID_книги```

Обновление данных о книге:  
```curl -X PUT URL/books/ID_книги -H "Content-Type: application/json" -d "{\"title\": \"Новое название\", \"author\": \"Автор\", \"isbn\": \"123456789\", \"year\": 2024}"```

Регистрация читателя:  
```curl -X POST URL/users -H "Content-Type: application/json" -d "{\"name\": \"Александр Петров\", \"email\": \"petrov@mail.ru\"}"```

Просмотр книг на руках у читателя:  
```curl URL/users/ID_читателя/books```

Выдача книги:  
```curl -X POST URL/issues -H "Content-Type: application/json" -d "{\"book_id\": \"ID_книги\", \"user_id\": \"ID_читателя\"}"```

Возврат книги:  
```curl -X POST URL/returns -H "Content-Type: application/json" -d "{\"book_id\": \"ID_книги\", \"user_id\": \"ID_читателя\"}"```
